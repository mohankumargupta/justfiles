# Justfiles

Personal cross-platform(linux-amd64, pi-aarch64, windows-x86_64 ) justfiles.

## Installing Software

#### Windows

Use winget, if not available, then scoop.

#### Linux

Works on all variants(x86_64, aarch64)

Use nixos packages

Usually, eg. yt-dlp

```sh
nix-env -iA nixpkgs.yt-dlp
```
